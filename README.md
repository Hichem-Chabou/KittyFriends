# KittyFriends
Web Interface made with React  

## Demo  
Here's a live demo: https://kittyfriends.surge.sh/

## Installation
1. clone the repository
2. go to the directory
3. open the terminal and type: <br>
`
$ npm install
`

## Usage
```bash
$ npm start
```

Then go to http://localhost:3000/ to view the app
